# Weather Desktop Application

A desktop weather application built using Python and Tkinter.

## Features
- Search weather by city
- Real-time weather data using OpenWeatherMap API
- Current local time of the searched city
- Temperature, humidity, pressure, wind speed
- User-friendly graphical interface

## Technologies Used
- Python
- Tkinter
- OpenWeatherMap API
- Geopy
- TimezoneFinder
- Requests

## Learning Outcomes
- API Integration
- GUI Development
- JSON Parsing
- Error Handling
